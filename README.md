# 預覽圖片畫面

提供一系列婚禮照片供使用者下載

### 使用方式  

把圖片丟進 `images` 資料夾中即可  

執行 `node ./src/utils/sharp.js ./src` => 圖片壓縮

---

### 安裝方式  

`git clone {專案}` 

`cd {專案資料夾}` 

`npm i` 

`npm run dev` : local模式： http://localhost:3000 可以看到畫面 

`npm run build` : 產生正式環境的程式

`npm run start` : 執行指令
